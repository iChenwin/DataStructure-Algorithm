1. 线性表
  1. 顺序存储结构  
    顺序存储结构就是两个相邻的元素在内存中也是相邻的。  
    优点是查询的时间复杂度为O(1)  
    缺点是插入和删除的时间复杂度最坏能达到O(n)
  2. 链式存储结构  
    优点是插入和删除的时间复杂度为O(1)  
    缺点是访问的时间复杂度最坏为O(n)
2. 链表  
  链表就是链式存储的线性表。  
  根据指针域的不同，链表分为`单向链表`、`双向链表`、`循环链表`等等。
  1. 单向链表(slist)  
    每个元素包含两个域，`值域`和`指针域`，我们把这样的元素称之为`节点`。  
    每个节点的指针域内有一个指针，指向下一个节点，而最后一个节点则指向一个空值。
